Na tela de "Criar Conta" o utilizador terá de preencher corretamente todos os campos e no final para confirmar os dados basta clicar no botão de "Criar".

O utilizador tem que inserir: 

- Nome
- Sobrenome
- Email
- Password
- Numero de telemovel
